# E-commerce Customer Behavior Analysis
**Prepared for:** Npontu Technologies  
**Role:** Intelligent Systems Services Engineer  

## Overview
Analysis of 50,000 synthetic e-commerce customers to derive
actionable business insights using machine learning.

## Key Results
- Model: Gradient Boosting Classifier
- AUC: 0.714
- High-risk customers identified: 1,249
- Revenue at risk: $105,885

## Tools Used
- Python, Pandas, Scikit-learn, SHAP
- Plotly, Matplotlib, Seaborn
- Apache Kafka (simulated)
